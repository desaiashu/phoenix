# Phoenix

Phoenix is a controller for audio reactive lighting

Hardware:
- Feather M4 Express
- Featherwing OLED screen
- TSL2591 Light Sensor
- MEMS PDM Mic
- 3 strips of 72 WS2812 LEDs (216 total LEDs)

Software:
- Circuitpython
- OLED lib
- Neopixel lib
- Custom LED control lib
